Create new Details View

Create a new view in views.py file, that will handle incoming requests to /details/ url:

my_tennis_club/members/views.py :

{# Screenshot image of details link in all_members.html file #} {# ./tested_images/details_views.html #}

Details view does following:

  • Gets an id as an argument.
  • Uses an id to locate a correct record in Member table.
  • Loads details.html template.
  • Creates an object containing a member.
  • Sends an object to details.html template
  • Outputs (Displays result of) HTML that is rendered by details.html template.

Add URLS
