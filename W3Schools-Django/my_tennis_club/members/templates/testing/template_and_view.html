Modify View

Next we need to make model data available in template and this is done in a view.

In a view we have to import Member model and send it to a template, as shown in image below:

my_tennis_club/members/views.py :

{# Screenshot image of views.py code for members view #}

The members view does the following:

  • Creates a mymembers object with all values of Member model.
  • Loads all_members.html template
  • Creates an object containing mymembers object.
  • Sends an object to a template.
  • Outputs HTML that is rendered by template.
