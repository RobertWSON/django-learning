Django Add Master Template

Extends Tag

In previous pages we created two templates, one for listing all members and one for details about a member.

These templates have a set of HTML code that are same for both templates.

Django provides a way of making a "parent template" that you can include in all pages to do stuff that is same in all pages.

Start by creating a template called master.html, with all necessary HTML elements:

Master

my_tennis_club/members/templates/master.html :

{# Screenshot image of code for master.html template setup #} {# ./tested_images/master_template.html #}

Do you see Django block Tag inside title element and body element ?

They are placeholders, telling Django to replace this block with content from other sources.

Modify Templates

Now two templates (all_members.html and details.html) can use this master.html template.

This is done by including master template with { % extends % } tag and inserting a title block and content block:
