QuerySet Descending Order

By default, a result is sorted ascending (lowest value first).
To change direction to descending (highest value first), use minus sign - in front of a field name :

QuerySet Order By Method with Descending Order Example

Display firstname field in descending order :

Queryset object :

{{ mymembers_descend }}

Loop through items :

{% for x in mymembers_descend %} {% endfor %}
ID Firstname Lastname
{{ x.id }} {{ x.firstname }} {{ x.lastname }}
