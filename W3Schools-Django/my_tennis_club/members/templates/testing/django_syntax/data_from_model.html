Django Data from a Model Example



    {% for x in mymembers %}
  • {{ x.firstname }}
    • {% endfor %}

This Django list is using mymembers variable from views.py, in a for loop {% for x in mymembers %}.
An x object is used to represent each record for a member from our model (table) of our database.
When we use <li>{{ x.firstname }}</li> , we are getting each member object
through using mymembers = Member.objects.all().values() in views.py and showing a value for each firstname property in a list.
The result as shown above is firstname value for each of 5 members in an unordered list.
