Django Variable Created in Template Example



{% with firstname="Tobias" %}

Hello {{ firstname }}, how are you?

{% endwith %}


This Django variable is created in template using a {{ with firstname="Tobias" }} template tag, where
a variable is created and a value is given.
Variable is shown in h1 heading using {{ firstname }} and with template tag is ended using {% endwith %} .

Remember a Django Variable is created here using HTML template and we jave not needed to get it from a view in view.py file.
