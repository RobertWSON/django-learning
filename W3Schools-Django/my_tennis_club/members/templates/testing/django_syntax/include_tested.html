Django Include Tag

Include

The include tag allows you to include a template inside current template.

This is useful when you have a block of content that is the same for many pages.

Include with Footer Example

Hello

This page contains a footer in a template.

{% include 'testing/footer.html' %}

Check out two templates to see what they look like, and views.py to see reference to child template.

Variables in Include

{#

Variables in Include

#} {#

You can send variables into template using with keyword.

#}

You can send variables into a template by using with keyword.

In include file, you refer to variables by using { { variablename } } syntax:

Include with Variables Example

{# W3Schools template below used for this Include with Variables example #} {% include "testing/mymenu.html" with me="TOBIAS" sponsor="W3SCHOOLS" %}

Welcome

This is my web site.

Check out mymenu.html to see the HTML content of the include.
