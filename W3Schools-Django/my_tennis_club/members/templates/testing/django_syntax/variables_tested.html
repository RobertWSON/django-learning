Normally, most of data you want to use in a template, comes from a model (table).

{# Image showing model data is used in HTML Template with a for loop in an unordered list#} #}

We use Django template tag {% for %} to loop through members from member model (table) in database.
Within unordered list, you can see that our variable from views.py is in a for loop and x is used to
represent each record id for a member, from model (table) in database.
x.firstname is using each id of a record to get a firstname for each record, so this means that we show a list with First Name for 5 members.
