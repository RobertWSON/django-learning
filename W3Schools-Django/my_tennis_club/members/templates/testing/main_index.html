Main Index Page

Our project needs a main page.

Main page (usually referred to as a home page) will be a landing page when someone visits root (main) folder of project.
Now, you can see below that there is an error when visting root folder 127.0.0.1:8000 of this project:

{# Screenshot image of error on main (home page) #} {# ./tested_images/Project Root Error - Main Page.png #}

This error is happening because we have not created a main page yet and debug is not set to true, which will be discussed on next
Django 404 Template page.
