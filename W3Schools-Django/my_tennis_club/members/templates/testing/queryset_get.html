Django QuerySet - Get Data

Get Django Data

There are different methods to get data from a model into a QuerySet.

values() Method

values() method allows you to return each object as a Python dictionary, with names and values as key/ value pairs :

values() Method Example

Queryset object :

{# {{ mymembers.mydata }} #} {# {{ mymembers }} #} {{ mymembers_getvals }}

Loop through items :

{# {% for x in mymembers.mydata %} #} {# {% for x in mymembers %} #} {% for x in mymembers_getvals %} {% endfor %}
ID Firstname Lastname
{{ x.id }} {{ x.firstname }} {{ x.lastname }}

Display Specific Rows

You can filter a search to only display specific rows/ records, by using filter() method.

Display Specific Rows Example

Returns only records where firstname is 'Emil'

Queryset object :

{# {{ mymembers.mydata3 }} #} {{ mymembers_getrows }}

Loop through items :

{# {% for x in mymembers.mydata3 %} #} {% for x in mymembers_getrows %} {% endfor %}
ID Firstname Lastname
{{ x.id }} {{ x.firstname }} {{ x.lastname }}
