linebreaks Example

Convert line breaks into < br >, and double line breaks into < p>:

Without linebreaks filter:

{{ mytext }}

With linebreaks filter:

{{ mytext|linebreaks }}

Right-click in this frame and check out frame source to see added HTML elements.

In views.py you can see what mytext variable looks like.

Definition and Usage

The linebreaks filter replaces line breaks with < br > and double line breaks with < p >.

See also linebreaksbr filter

Syntax

{#{ { value|linebreaks } }#}

{ { value|linebreaks } }

Template filters are defined by using a pipe | character followed by name of filter.
