Django Template Tag Reference

Template Filter - timesince

Timesince between datetimes Example

Return difference between two datetimes:

{{ mybirthdate|timesince:mydate }}

As you can see from this timesince example, we are using a django command to take time difference between a birth date and current date.

In views.py you can see what mybirthdate and mydate variables look like.

Definition and Usage

The timesince filter returns difference between two date times.

If you leave out date in argument, current date (now) will be used:

Syntax

{ { datetime|timesince:comparedate } }

Template filters are defined by using a pipe | character followed by name of filter.

Arguments are defined by using a colon : character followed by argument value.

Arguments

Value Description
comparedate Optional. A datetime you want to conmpare with. Default value is datetime now.
