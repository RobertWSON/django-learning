First item in a list Example

Returns first item in a list:

{{ fruits|first }}

As you can see from this first example, we are using a django command, to display first item from a list in an array for an object.

In views.py you can see what fruits variable looks like.

Definition and Usage

The first filter returns first item of an object

For string, the first filter returns first character.

First string character Example

Return first character of a string:

Use last filter to return last item.

Syntax

Template filters are defined by using a pipe | character followed by name of filter.

{ { value|first } }
