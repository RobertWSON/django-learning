Django Filter Tag Reference

Template Filter - slice

Slice Example

Get items 1, 2 and 3 from a list (first item is 0):

{{ fruits|slice:"1:4" }}

In views.py you can see what name variable looks like.

Definition and Usage

The slice filter returns specified items from a list.

You can define from (included) and to (not included) indexes.

If you use negative indexes, it means slicing from end of a list, -1 refers to last item, -2 refers to second last item etc.

Syntax

{ { list | slice"from:to" } }

Template filters are defined by using a pipe | character followed by name of filter.

Arguments are defined by using a colon : character followed by argument value.

Arguments

{##} {#
Value Description
fromfrom Optional. Index to start slicing. If omitted, default value is 0 and slicing will start at first item.
to#} to Optional. Index to end slicing. If omitted, slicing will end at last item.
