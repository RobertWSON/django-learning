Django Filter Tag Reference

Template Filter - title

Title Example

Convert a text into using all capitalized words:

{{ mytext|title }}

In views.py you can see what mytext variable looks like.

Definition and Usage

The title filter returns a value with all words converted into having first letter in upper case and rest of word in lower case.

Syntax

Template filters are defined by using a pipe | character followed by name of filter.

{ { value|title } }
