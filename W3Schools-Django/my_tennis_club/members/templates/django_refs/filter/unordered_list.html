Unordered_list Template Tag

Unordered_list Example

Convert Python list into an HTML list:

    {{ fruits|unordered_list }}

As you can see from this array example, we are using a django command to convert a django or python list from an array in an object, into an HTML list.

Definition and Usage

The unordered_list filter converts an object into an HTML list.

The start < ul> and end < /ul> is not included, but if you have a list inside a list, unordered_list filter will add proper < ul> and < /ul> for nested lists:

Syntax

template filters are defined by using a pipe | character followed by name of filter.

{# Need space below between a curly brace { and %, so that it can be displayed as text and does not think it is code #} { % value|unordered_list % }
