Django Filter Tag Reference

Template Filter - make_list

Make_list Example

Convert a string into a list:

{{ name|make_list }}

In views.py you can see what name variable looks like.

Definition and Usage

The make_list filter converts a value into a list.

Each character in original value becomes a list item.

Syntax

Template filters are defined by using a pipe | character followed by name of filter.

{ {value|make_list} }
