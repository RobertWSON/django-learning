Cut Example to Remove Several Letters

Remove phrase "snes" from name "Emil Refsnes":

{{ name|cut:"snes" }}

In views.py you can see what name variable looks like. The same cut view and variable are used for both examples.

Definition and Usage

The cut filter removes specified phrase from a value.

The cut filter is case sensitive.

All occurences of specified phrase/ character are removed.

Cut Example to remove one letter

{{ name|cut:"e" }}

Syntax

{ { value|cut:phrase } }

Template filters are defined by using a pipe | character followed by name of filter.

Arguments are defined by using a colon : character followed by argument value.
