Django Template Tag Reference

Template Filter - length_is

length_is for list items Example

Check if there are 3 items in a list:

{{ fruits|length_is:3 }}

As you can see from this length_is example, we are using a django command, which acts like a boolean, to determine if it is true that an object has 3 list items in an array.

In views.py you can see what fruits variable look like.

Definition and Usage

The length_is filter returns Trueif length of an object is same as specified number.

This can be useful when you are dealing with if statements:

Syntax

{ { value|length_is:len } }

Template filters are defined by using a pipe | character followed by name of filter.

Arguments are defined by using a colon : character followed by argument value.

Arguments

Value Description
len Required. A number to check if object's length is same as.
