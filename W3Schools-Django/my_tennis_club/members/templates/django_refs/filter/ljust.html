Django Filter Tag Reference

Template Filter - ljust

Ljust Example

Write a name, justified to left of a character field:

{{ name|ljust:20 }}

In HTML display does not show more than one space, but if you view page source you will see that there are 20 characters between <h1> and </h1> .

Or, to demonstrate we can use a PRE element to see that "Tobias" is left aligned:

{{ name|ljust:20 }} Refsnes

In views.py you can see what name variable looks like.

Definition and Usage

The ljust filter places a value to left in a field of specified length.

Syntax

{ { value|ljust:length } }

Template filters are defined by using a pipe | character followed by name of filter.

Arguments are defined by using a colon : character followed by argument value.

Arguments

Value Description
length Required. A number specifying length of a field where a value will be left justified in.
