Django Filter Tag Reference

Template Filter - dictsortreversed

Dictsortreversed Example

Sort a dictionary descending by year:

{% for x in cars|dictsortreversed:"year" %}

{{x.year}} {{x.brand}} {{x.model}}.

{% endfor %}

In views.py you can see what cars variables in an array looks like.

Definition and Usage

The dictsortreversed filter sorts a dictionary by a specified field, descending.

Use dictsort filter to sort ascending.

Syntax

{ { value|dictsortreversed:fieldname } }

Template filters are defined by using a pipe | character followed by name of filter.

Arguments are defined by using a colon : character followed by argument value.

Arguments

Value Description
fieldname Required. The name of the field to be sorted on.
