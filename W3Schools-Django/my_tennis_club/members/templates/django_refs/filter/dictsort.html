Dictsort Example

Sort a dictionary by year:

{% for x in cars|dictsort:"year" %}

{{x.year}} {{x.brand}} {{x.model}}.

{% endfor %}

In views.py you can see what cars variables in an array looks like.

Definition and Usage

The dictsort filter sorts a dictionary by a specified field.

The dictsort filter sorts ascending.

Use dictsortreversed filter to sort descending.

Syntax

{ { value|dictsort:fieldname } }

Template filters are defined by using a pipe | character followed by name of filter.

Arguments are defined by using a colon : character followed by argument value.

Arguments

{##}
Value Description
fieldnamefieldname Required. The name of the field to be sorted.
