Django Filter Tag Reference

Template Filter - center

Center Example

Write name in middle of a 20 characters field:

{#

{{ name:center:20 }}

center:20 not working #}

{{ name }}

In HTML, display does not show more than one space, but if you view page source you will see that there are 20 characters between <h1> and </h1>.

Or, to demonstrate we can use a textarea to see that "Tobias" is center aligned:

{# center:20 not working #}

As you can see from this center example, we are using a django center command, to display and center a string name variable from an object inside an HTML textarea.

In views.py you can see what name variable looks like.

Definition and Usage

The center filter places value in center of a value of specified length.

Syntax

{ { value|center:length } }

Template filters are defined by using a pipe | character followed by name of filter.

Arguments are defined by using a colon : character followed by argument value.

Arguments

Value Description
length Required. A number specifying length of field where value will be in center.
