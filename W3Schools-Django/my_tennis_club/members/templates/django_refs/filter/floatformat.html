Template Filter - floatformat

Floatformat with 2 decimal places Example

Round a number to only two decimal places:

{{ mynumber|floatformat:2 }}

As you can see from this floatformat example, we are using a django command to round a number that has 6 decimal places down to 2 decimal places.

Definition and Usage

The floatformat filter rounds a floating-point number to a specified number of decimals, or one, if no decimal argument is specified.

If argument decimal value is a positive number, a value will be displayed with specified number of decimals, even if value is an integer:

Syntax

{ { value|floatformat:decimalsg } }

Template filters are defined by using a pipe | character followed by name of filter.

Arguments are defined by using a colon : character followed by argument value(s).

Arguments

Value Description
decimals Optional. A number specifying number of decimals. Default value is -1, which will round a number to only one decimal, or no decimals if number has no decimals.
g Optional. If present, a number will be displayed with a seperator for each thousand, using local settings (using , for English and . for Norwegian).
u Optional. If present, a number will be displayed with a seperator for each thousand, using local settings. If u is present g cannot be present.
