Addslashes Example

Escape the quote in the name variable:

{{ name|addslashes }}

In views.py you can see what the name variable looks like for both examples.

Definition and Usage

The addslashes filter adds a slash \ character before any quotes.

This can be helpful when you are dealing with JavaScript and you have to escape quote characters.

Addslashes Filter Example with Button and Alert Message

Escape the name, or else you will get an error when alerting it:

This example would return an error when someone clicked the button, if we did not escape the quote in the name variable.

In views.py you can see what the name variable looks like.

Without addslashes filter in example above, the page would get an error when someone clicked button.

Syntax

Template filters are defined by using a pipe | character followed by name of filter.

{ { value|addslashes } }
