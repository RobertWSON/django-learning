Django Filter Tags Reference

A List of all filter keywords:

Keyword Description
add Adds a specific value.
addslashes Adds a slash before any quote characters, to escape strings.
capfirst Returns first letter in uppercase.
center Centers value in middle of a specified width.
cut Removes any specified characters or phrases.
date Returns dates in specified format.
default Returns a specific value if value is false.
default_if_none Returns a specified value if value is none.
dictsort Sorts a dictionary by given value.
dictsortreversed Sorts a dictionary reversed, by given value.
divisibleby Returns True if value can be divided by specified number, otherwise it returns False.
escape Escapes HTML code from a string..
escapejs Escapes JavaScript code from a string.
filesizeformat Returns a number into a file size format.
first Returns the first item of an object (for Strings, the first character is returned).
floatformat Rounds floating numbers to a specified number of decimals, default one decimal.
force_escape Escapes HTML code from a string.
get_digit Returns a specific digit of a number.
iriencode Convert an IRI into a URL friendly string.
join Returns the items of a list into a string.
json_script Returns an object into a JSON object surrounded by script tags.
last Returns the last item of an object (for Strings, the last character is returned).
length Returns the number of items in an object, or the number of characters in a string.
length_is Returns the number of items in an object, or the number of characters in a string.
linebreaks Returns the text with br tag instead of line breaks, and p tag instead of more than one line break.
linebreaksbr Returns the text with br tag instead of line breaks.
linenumbers Returns the text with line numbers for each line.
ljust Left aligns the value according to a specified width.
lower Returns the text in lower case letters.
make_list Converts a value into a list object.
phone2numeric Converts phone numbers with letters into numeric phone numbers.
pluralize Adds a 's' at the end of a value if the specified numeric value is not 1.
pprint pprint is used for
random Returns a random item of an object.
rjust Right aligns the value according to a specified width.
safe Marks that this text is safe and should not be HTML escaped.
safeseq Marks each item of an object as safe and the item should not be HTML escaped.
slice Returns a specified slice of a text or object.
slugify Converts text into one long alphanumeric-lower-case word.
stringformat Converts value into a specified format.
striptags Removes HTML tags from a text.
time Returns a time in specified format.
timesince Returns difference between two datetimes.
timeuntil Returns difference between two datetimes.
title Upper cases first character of each word in a text, all other characters are converted to lower case.
truncatechars Shortens a string into specified number of characters.
truncatechars_html Shortens a string into specified number of characters and not considering length of any HTML tags.
truncatewords Shortens a string into specified number of words.
truncatewords_html Shortens a string into specified number of words, not considering any HTML tags.
unordered_list Returns items of an object as an unordered HTML list.
upper Returns text in upper case letters.
urlencode URL encodes a string.
urlize Returns any URLs in a string as HTML links.
urlizetrunc Returns any URLs in a string as HTML links, but shortens links into specified number of characters.
wordcount Returns number of words in a text.
wordwrap Wrap words at a specified number of characters.
yesno Converts Booleans values into specified values.
i18n i18n is used for
l10n l10n is used for
tz tz is used for
