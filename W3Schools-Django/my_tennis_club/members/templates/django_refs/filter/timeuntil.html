Template Filter - timeuntil

Timeuntil for an upcoming event Example

Return time until May 17, 2050:

{{ marslanding|timeuntil }}

As you can see from this timeuntil example, we are using a django command to work out time from a current date now until a mars landing event in 2050.

In views.py you can see what marslanding variable look like.

Definition and Usage

The timeuntil filter returns difference between two date times.

The default datetime to start count is now, but you can also specify it:

Syntax

{ { datetime|timeuntil:comparedate } }

Template filters are defined by using a pipe | character followed by name of filter.

Arguments are defined by using a colon : character followed by argument value.

Arguments

Value Description
comparedate Optional. A datetime you want to conmpare with. Default value is datetime now.
