Definition and Usage

The timeuntil filter returns difference between two date times.

The default datetime to start count is now, but you can also specify it:

{#

Syntax

{ { datetime|timeuntil:comparedate } }

Template filters are defined by using a pipe | character followed by name of filter.

Arguments are defined by using a colon : character followed by argument value.

Arguments

{##}
Value Description
comparedatecomparedate Optional. A datetime you want to conmpare with. Default value is datetime now.
