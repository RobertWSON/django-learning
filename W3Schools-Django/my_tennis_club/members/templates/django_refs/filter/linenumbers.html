Django Filter Tag Reference

Template Filter - linenumbers

Linenumbers Example

Add line number before each line:

{{ mytext|linenumbers }}

In views.py you can see what mytext variable looks like.

Definition and Usage

The linenumbers filter adds a line number before each line of a text.

Syntax

{ { value|linenumbers } }

Template filters are defined by using a pipe | character followed by name of filter.
