Lower Example

Display firstname with all lower case letters:

{{ firstname|lower }}

In views.py you can see what firstname variable looks like.

Definition and Usage

The lower filter returns a value in all lower case letters.

Syntax

Template filters are defined by using a pipe | character followed by name of filter.

{ { value|lower } }
