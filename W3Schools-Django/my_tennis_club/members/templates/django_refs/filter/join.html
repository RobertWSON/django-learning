Join Example

Join all items in a list into a string, using a hash character as a separator:

{{ fruits|join:"#" }}

In views.py you can see what fruits variable looks like.

These string items are taken from a list in an array for an object.

As you can see from this join example, we are using a django command that has a seperator. The seperator here is in the form of a # character and is used to join string items.

Definition and Usage

The join filter takes all items in an iterable and joins them into one string.

A string must be specified as a separator.

Syntax

{# { { value|join:separator } } #} { { value|join:separator } }

Template filters are defined by using a pipe | character followed by name of filter.

Arguments are defined by using a colon : character followed by argument value(s).
