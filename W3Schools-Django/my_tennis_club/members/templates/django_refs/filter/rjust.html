Django Filter Tag Reference

Template Filter - rjust

Rjust Example

Write name, justified to right of a 20 characters field:

{{ name|rjust:20 }}

In HTML display does not show more than one space, but if you view page source you will see that there are 20 characters between <h1> and </h1>.

or, to demonstrate we can use a PRE element to see that "Tobias is right aligned:

King {{ name|rjust:20 }} Refsnes

In views.py you can see what name variable looks like.

Definition and Usage

The rjust filter places a value to right in a field of specified length.

Syntax

{ { value|rjust:length } }

Template filters are defined by using a pipe | character followed by name of filter.

Arguments are defined by using a colon : character followed by argument value.

Arguments

Value Description
length Required. A number specifying length of a field where a value will be right justified in.
