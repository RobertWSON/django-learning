Add Example

Add 10 dollars to all prices:

{% for x in prices %}

The price is {{ x|add:"10" }} dollars.

{% endfor %}

In views.py you can see what prices variable looks like.

As you can see, this first example adds a price amount within a heading.
This price amount is taken from a prices variable in views.py and x is used
to represent a price as a character within a for loop.

Definition and Usage

The add filter adds a specified argument to a value.

The add filter tries to convert the values into numbers and return a sum, but if it is not possible to convert values
into numbers, a specified argument will be added at the end:

Add Filter Example with -CHECK

Add phrase "-CHECK" at end of each fruit:

{# add Check not working just yet #} {% for x in fruits %}

{{ x|add:"-CHECK" }}

{% endfor %}

Template filters are defined by using a pipe | character followed by the name of the filter.

Arguments are defined by using a colon : character followed by the argument value.

Arguments

{##}
Value Description
valuetoaddvaluetoadd Required. The value to add to the original value. If both values can be converted to
numbers, the sum will be returned, otherwise the added value will be added at the end of
original value.
