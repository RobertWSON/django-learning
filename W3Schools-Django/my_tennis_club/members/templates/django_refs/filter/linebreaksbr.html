Linebreaksbr Example

Convert any line breaks into <br>:

Without linebreaksbr filter:

{{ mytext }}

With linebreaksbr filter:

{{ mytext|linebreaksbr }}

Right-click in this frame and check out the frame source to see the added <br> elements.

In views.py you can see what the mytext variable looks like.

Definition and Usage

The linebreaksbr filter replaces line breaks with <br> elements.

The difference between this filter and linebreaks filter is that this filter replaces double line breaks with two <br>
elements, but the linebreaks filter replaces double line breaks with a <p> element.

Syntax

{ { value|linebreaksbr } }

Template filters are defined by using a pipe | character followed by name of filter.
