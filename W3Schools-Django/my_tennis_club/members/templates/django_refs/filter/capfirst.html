Django Filter Tag Reference

Template Filter - capfirst

Capfirst Example

Write an animal as a header (with first letter in upper case):

{{ animal|capfirst }}

In views.py you can see what the animal variable looks like.

Definition and Usage

The capfirst filter capitalizes first letter of a value.

Syntax

Template filters are defined by using a pipe | character followed by name of filter.

{ { value| capfirst } }
