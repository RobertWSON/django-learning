Django Filter Tag Reference

Template Filter - random

Random Example

Return a random list item:

My favourite fruit is {{ fruits|random }}

In views.py you can see what fruits variable looks like.

Try refreshing example to see fruit changes.

Definition and Usage

The random filter returns a random item of an object.

Examples:

Syntax

Template filters are defined by using a pipe | character followed by name of filter.

{ { value|random } }
