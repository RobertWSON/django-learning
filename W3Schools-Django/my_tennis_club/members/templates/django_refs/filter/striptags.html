Striptags Template Tag

Striptag Example

Remove HTML from text:

Without striptags filter:

{{ mytext }}

With striptags filter:

{{ mytext|striptags }}

As you can see from this text example, without striptags you have html tags shown with text and with striptags, html tags are not shown, so just text is shown.

In views.py you can see what mytext variable looks like.

Definition and Usage

The striptags filter removes any html tags from a value.

Syntax

{# Need space below between a curly brace { and %, so that it can be displayed as text and does not think it is code #} { % value|striptags % }

template filters are defined by using a pipe | character followed by name of filter.
