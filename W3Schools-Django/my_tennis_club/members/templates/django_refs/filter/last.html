Django Template Tag Reference

Template Filter - last

Last for an Object Example

Return number of items in a list:

{{ fruits|last }}

As you can see from this last example, we are using a django command, to display the last fruit item taken from a list in an array for an object.

In views.py you can see what fruits variable look like.

Definition and Usage

The last filter returns last item of an object.

For strings, last filter returns last character:

Syntax

Template filters are defined by using a pipe | character followed by name of filter.

{ { value|last } }
