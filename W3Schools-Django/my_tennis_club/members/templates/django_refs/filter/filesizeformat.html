Django Filter Tag Reference

Template Filter - filesizeformat

Filesizeformat Example

Display file size in a more readable format:

{{ size|filesizeformat }}

In views.py you can see what the size variable looks like.

Definition and Usage

The filesizeformat filter converts a large number into a more human readable.

Examples:

  • 1024 is converted into 1.0 KB.
  • 524288 is converted into 512.0 KB.

Syntax

Template filters are defined by using a pipe | character followed by name of filter.

{ { value|filesizeformat } }
