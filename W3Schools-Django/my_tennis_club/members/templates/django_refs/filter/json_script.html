Definition and Usage

The json_script filter converts an object, like a python tuple, into a JSON object surrounded by <script> </script> tags. {# < script > < / script > tags. #}

Syntax

{# { { value|json_script:id } } #} {#This maybe better with code#}

{ { value|json_script:id } }

Template filters are defined by using a pipe | character followed by name of filter.

Arguments are defined by using a colon : character followed by argument value(s).

Arguments

{##} {##}
Value Description
ididdigitRequired. A string that will be an id value of a script element.
