Template Filter - json_script

Json_script Example

Convert an object into a JSON object:

{{ cars|json_script:"mycars" }}

Right-click in this frame and check out frame source to see added <script> element.

In views.py you can see what cars variable looks like.

Definition and Usage

The json_script filter converts an object, like a python tuple, into a JSON object surrounded by <script> </script> tags. {# < script > < / script > tags. #}

Syntax

{# { { value|json_script:id } } #} This maybe better with code { { value|json_script:id } }

Template filters are defined by using a pipe | character followed by name of filter.

Arguments are defined by using a colon : character followed by argument value(s).

Arguments

{##}
Value Description
iddigitRequired. A string that will be an id value of a script element.
