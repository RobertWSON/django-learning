Django Filter Tag Reference

Template Filter - escape

Escape Example

Display a text containing HTML, with and without escape:

{% autoescape off %}

{{ heading|escape}}

{{ heading}}

{% endautoescape %}

In views.py you can see what heading variable looks like.

Note: Escaping HTML characters is a default setting in DJango, so we have to turn off autoescape in example to see difference.

Definition and Usage

The escape filter escapes HTML characters from a value.

Note: Escaping HTML characters is a default setting in Django, so we have to turn off autoescape in example to be able to see difference.

These characters are escaped:

  • < is converted to <
  • > is converted to >
  • ' is converted to '
  • " is converted to "
  • & is converted to &

Syntax

Template filters are defined by using a pipe | character followed by name of filter.

{ { value|default:defaultvalue } }
