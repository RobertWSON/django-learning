Default_if_none Example

Write color names, but write "nocolor" if value is none:

{% for x in colors %}

{{ x|default_if_none:"nocolor" }}

{% endfor %}

In views.py you can see what colors variable look like.

Definition and Usage

The default_if_none filter allows you to specify a default value to use if value is None.

The value has to be exactly None for this filter to have an effect.

Syntax

{ { value|default_if_none:defaultvalue } }

Template filters are defined by using a pipe | character followed by name of filter.

Arguments are defined by using a colon : character followed by argument value.

Arguments

{##}
Value Description
defaultvaluedefaultvalue Required. The value to return if value is None.
