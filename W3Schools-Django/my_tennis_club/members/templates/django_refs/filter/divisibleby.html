Divisibleby 5 Example

Check if a value is divisible by 5:

{% if totalsum|divisibleby:5 %}

Yes, it is!

{% endif %}

In views.py you can see what totalsum variable looks like for both examples.

Definition and Usage

The divisibleby filter returns True if a value is divisible by an argument, otherwise it
returns False.

Divisibleby 3 Example

Display an answer, is totalsum divisibleby 3:

{{ totalsum|divisibleby:3 }}

Syntax

{ { value|divisibleby:num } }

Template filters are defined by using a pipe | character followed by name of filter.

Arguments are defined by using a colon : character followed by argument value.

Arguments

{##}
Value Description
numnum Required. A number to try to divide a value with.
