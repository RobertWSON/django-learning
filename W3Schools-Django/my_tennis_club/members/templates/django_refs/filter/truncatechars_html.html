Definition and Usage

The truncatechars_html filter truncate a value if it is longer than a specified number of characters.

The truncatechars_html filter will not count any HTML tags, but closes tags that are being started, but not ended.

If a value gets truncated, a value will add three dots (...) at end of a string.

See also:

{##} truncatechars
{##} {##} truncatewords
{##} {##} truncatewords_html {##}

Syntax

{ { value|truncatechars_html:number } }

Template filters are defined by using a pipe | character followed by name of filter.

Arguments are defined by using a colon : character followed by argument value.

Arguments

{##}
Value Description
numbernumber Required. A number specifying how many characters are allowed.
