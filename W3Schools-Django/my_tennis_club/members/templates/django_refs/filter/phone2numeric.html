Phone2numeric Example

Display actual phone number:

{{ phone|phone2numeric }}

In views.py you can see what phone variable looks like.

Definition and Usage

The phone2numeric filter converts any letter in a phone number like into actual numeric phone number.

E.g. "555-automobile" becomes 555-288662453.

Syntax

Template filters are defined by using a pipe | character followed by name of filter.

{ { value|phone2numeric } }
