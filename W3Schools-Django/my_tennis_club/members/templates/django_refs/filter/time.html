Django Filter Tag Reference

Template Filter - time

Time Example

Return time from a date object:

{{ mydate|time }}

As you can see from this time example, we are using a django command, to display a time in 24 hour format with hour and minutes.

In views.py you can see what mydate variable looks like.

Definition and Usage

The time filter formatsand returnsa time from a datetime object.

You can specify format by using these format characters.

g Hour ( 1-12 )
G Hour ( 0-23 )
h Hour ( 01-12 )
H Hour ( 00-23 )
i Minutes ( 00-59 )
s Seconds ( 00-59 )
u Microseconds ( 000000-999999 )
a am/pm suffix ( a.m. or p.m. )
A AM/PM suffix ( A.M. or P.M. )
f Time, hour and minute, but minutes are left out if they are 0
P Same as f, but also including a.m. or p.m. and:
If time is exactly "12:00 a.m." it returns "midnight".
If time is exactly "12:00 p.m." it returns "noon".

Syntax

{ { datetime|time:"format"} }

Template filters are defined by using a pipe | character followed by name of filter.

Arguments are defined by using a colon : character followed by argument value.

Arguments

Value Description
format Optional. The format you want to return time in. Default value TIME_FORMAT, which will be same as "H:i"..
