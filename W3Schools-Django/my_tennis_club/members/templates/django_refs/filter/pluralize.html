Django Filter Tag Reference

Template Filter - pluralize

Pluralize number with 's' Example

Add a 's' at end of number, if number is not 1:

{% for x in arr %}

You have {{ x }} life{{ x|pluralize }} left.

{% endfor %}

In views.py you can see what arr variable looks like for all examples.

Definition and Usage

The pluralize filter adds a 's' suffix if value is not 1.

You can specify another suffix with suffix argument:

Pluralize with another suffix Example

{% for x in arr %}

You have {{ x }} kiss{{ x|pluralize:'es' }} left.

{% endfor %}

For words with different single and plural endings, you can add both by seperating them by a comma:

Pluralize for words with single and plural endings Example

{% for x in arr %}

You have {{ x }} butterfl{{ x|pluralize:'y,ies' }} left.

{% endfor %}

Syntax

{ { value|pluralize:suffix } }

Template filters are defined by using a pipe | character followed by name of filter.

Arguments are defined by using a colon : character followed by argument value.

Arguments

Value Description
suffix Optional. The suffix to add to a value.
Default 's'.
For words with different single and plural endings, you can add both by seperating them by a comma.
