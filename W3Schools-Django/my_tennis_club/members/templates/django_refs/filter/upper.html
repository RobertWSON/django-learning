In HTML Template we are using an upper filter on firstname variable to show a value as an H1 heading.

Definition and Usage

The upper filter returns a value in all upper case letters.

See also lower filter for showing lower case letters.

Syntax

Template filters are defined by using a pipe | character followed by name of filter.

{#{ { value|upper } }#} { { value|upper } }
