Definition and Usage

The truncatewords filter truncate a value if it is longer than a specified number of words.

If a value gets truncated, a value will add three dots (...) at end of a string.

See also:

{##} truncatewords_html
{##} {##} truncatechars
{##} {##} truncatechars_html {##}

Syntax

{ { value|truncatewords:number } }

Template filters are defined by using a pipe | character followed by name of filter.

Arguments are defined by using a colon : character followed by argument value.

Arguments

{##}
Value Description
numbernumber Required. A number specifying how many words are allowed.
