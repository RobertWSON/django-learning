Django Filter Tag Reference

Template Filter - get_digit

Get_digit Example

Return second digit, counting from end:

{{ mynumber|get_digit:2 }}

As you can see from this get_digit example, we are are using a django command, to get second number digit from end for a variable in an object.

In views.py you can see what mynumber variable looks like.

Definition and Usage

The get_digit filter returns specified digit from a number, counting from end of a number.

Syntax

{# { { value|get_digit:digit } } #} { { value|get_digit:digit } }

Template filters are defined by using a pipe | character followed by name of filter.

Arguments are defined by using a colon : character followed by argument value(s).

Arguments

{##}
Value Description
digitdigitRequired. A number specifying what digit to return. 1 returns last number, 2 returns second to last etc.
