Django Template Tag Reference

Template Filter - length

length for an Object Example

Return number of items in a list:

{{ fruits|length }}

As you can see from this length example, we are using a django command, to display number of items from a list in an array for an object.

In views.py you can see what fruits variable look like.

Definition and Usage

The length filter returns number of items in an object.

For strings, length filter returns number of characters:

Syntax

Template filters are defined by using a pipe | character followed by name of filter.

{ { value|length } }
