Django Filter Tag Reference

Template Filter - default

Default Example

Write color names, but write "nocolor" if value is empty:

{% for x in colors %}

{{ x|default:"nocolor"}}

{% endfor %}

In views.py you can see what colors variable looks like.

Definition and Usage

The default filter allows you to specify a default value to use if value evaluates to False.

Values that evaluate to False are:

  • Empty Strings ""
  • Any numeric 0
  • Any empty object [], (), {}, set(), range(0)
  • None
  • False

Syntax

{ { value|default:defaultvalue } }

Template filters are defined by using a pipe | character followed by name of filter.

Arguments are defined by using a colon : character followed by argument value.

Arguments

Value Description
defaultvalue Required. The value to return if value is None.
