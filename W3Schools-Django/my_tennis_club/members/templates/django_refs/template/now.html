Django Template Tag Reference

Now Template Tag

Now - Current Date

Now - Current Date Example

Add current date:

{% now "Y-m-d "%}

Definition and Usage

The now tag inserts current date and/or time, according to specified format.

Now - Current Date and Time

Now - Current Date and Time Example

Add current date and time:

{% now "Y-m-d G:i:s" %}

Syntax

{ % now format % }

Parameters

Value Description
count Optional. The number of words or paragraphs to display. Default 1.
method "Optional. Specifies whether to display blocks, words or HTML paragraphs."
b"for blocks of text"
p"for HTML Paragraphs"
p"for words"
"Default "b
random "Optional. If specified, random text start at a random word of "lorum ipsum" text"
