Django Template Tag Reference

Lorem Template Tag

Lorem Random Text

Lorem Random Text Example

Insert 50 words of random text:

{% lorem 50 w %}

Check out views.py to see what the heading variable looks like.

Definition and Usage

The lorem tag iserts a specified amount of random text.

The "random" text is famous "Lorem ipsum" text, in lower case letters.

Lorem Paragraphs

Lorem Paragraphs Example

Insert 5 paragraphs of random text:

{% lorem 5 p %}

Syntax

{# Need space below between a curly brace { and %, so that it can be displayed as text and does not think it is code #} { % lorem count method random % }

Parameters

Value Description
count Optional. The number of words or paragraphs to display. Default 1.
method "Optional. Specifies whether to display blocks, words or HTML paragraphs."
b"for blocks of text"
p"for HTML Paragraphs"
p"for words"
"Default "b
random "Optional. If specified, random text start at a random word of "lorum ipsum" text"
