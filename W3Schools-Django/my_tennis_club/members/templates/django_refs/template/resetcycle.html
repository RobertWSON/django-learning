Django Template Tag Reference

Resetcycle Template Tag

Reset a Cycle on a List Item

Resetcycle on a List Item Example

Reset the cycle if the fruit is "Banana":

    {% for x in fruits %}
  • {{ x }}
    • {% if x == "Banana" %} {% resetcycle %} {% endif %} {% endfor %}

As you can see from this list example, Apple is red and Banana is green color.
Cycle then restarts and gets reset again, so next item Cherry is red and finally Orange fruit item is green color.

Check out views.py to see what fruits variable looks like, for both examples.

Definition and Usage

The resetcycle tag is used inside a cycle and resets a cycle, making it start at beginning again.

It does not reset loop, only the cycle.

If you have multiple cycles, you can specify which one to reset with name argument:

Reset a Cycle from Multiple Cycles

Resetcycle from Multiple Cycles Example

Reset mybg cycle if the fruit is "Banana":

    {% for x in fruits %}
  • {{ x }}
    • {% if x == "Banana" %} {% resetcycle mybg %} {% endif %} {% endfor %}

As you can see from this list example, we are resetting a cycle named mybg for background color.
Apple has grey and Banana has beige background color.
Cycle then gets reset again, so next item Cherry has grey and finally Orange fruit item has beige background color.

Syntax

{# Need space below between a curly brace { and %, so that it can be displayed as text and does not think it is code #} {#{ % resetcycle name % } #}

Parameters

Value Description
name Optional. The name of a cycle to reset.
