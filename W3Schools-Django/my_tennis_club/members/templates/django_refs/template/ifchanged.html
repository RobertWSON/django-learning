Definition and Usage

The ifchanged tag allows you to check a value in a loop and output code if a value has changed
since last iteration.
If an iteration object has many values per iteration, you can specify which value to check and a block of code
will only display if a value has changed since last iteration:

Ifchanged Array Property Example

Loop through members object and check if a brand property has changed:

{% for x in cars %} {% ifchanged x.brand %}

{{ x.brand }}

{% endifchanged %}

{{ x.model }}, {{ x.year }}

{% endfor %} {# Show image of views.py and urls.py file #} {# #} {# Explain code in views.py file #}

You can also define a { % else % } clause for content that should be displayed if a value has not changed:

Ifchanged List Messages Example

Loop through a list and write different messages if a value has changed or not:

{% for x in mylist %} {% ifchanged %}

New value: {{ x }}

{% else %}

Same value: {{ x }}
