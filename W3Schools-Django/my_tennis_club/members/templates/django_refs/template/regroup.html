Django Template Tag Reference

Regroup Template Tag

Regroup from an Array

Regroup cars by Brand Example

Display all cars with a new header for each brand:

{% regroup cars by brand as newlist %} {{ newlist }} {% for x in newlist %}

{{ x.grouper }}

{% for y in x.list %}

{{ y.model }}: {{ y.year }}

{% endfor %} {% endfor %}

As you can see from this array example, each car has been grouped, displaying it's model and year.
Note that it is in the same order as the array, but just those 2 properties values have been shown.

Check out views.py to see what fruits variable looks like, for both examples.

Definition and Usage

Parameters

Value Description
object Required. A list or object that you want to
regroup.
object.property Required. The name of the property you
want to group by.
newname Required. A new name for a returned object
