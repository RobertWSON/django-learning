Django Template Tag Reference

Verbatim Template Tag

verbatim without execute Example

{% verbatim %} {% for x in fruits %}

{{ x }}

{% endfor %} {% endverbatim %}

Definition and Usage

The verbatim template tag is used to stop Django from executing code.

Anything between { % verbatim % } and { % endverbatim % } will not be executed, but rendered as output instead.

To be sure that you refer to correct verbatim code block, you can add a name to it:

Verbatim choosing from more than one Example

{% verbatim mycode %} {% for x in fruits %} {% verbatim %}

{{ x }}

{% endverbatim %} {% endfor %} {% endverbatim mycode %}
