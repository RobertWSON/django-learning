In HTML Template we are using verbatim Django command to show just Django code between { % verbatim % } and { % endverbatim % } commands.

For this example we show Django commands { % for x in fruits % } and { % endfor % } and just Django code { x } between HTML h1 heading tags.

Using verbatim means that we actually are not using data from our views.py file, but showing Django code used for output.

Definition and Usage

The verbatim template tag is used to stop Django from executing code.

Anything between { % verbatim % } and { % endverbatim % } will not be executed, but rendered as output instead.

To be sure that you refer to correct verbatim code block, you can add a name to it:

{#

Verbatim choosing from more than one Example

When there can be more than one verbatim tag in a code block, you can use a name argument to specify which one you mean.

{% verbatim mycode %} {% for x in fruits %} {% verbatim %}

{{ x }}

{% endverbatim %} {% endfor %} {% endverbatim mycode %}
