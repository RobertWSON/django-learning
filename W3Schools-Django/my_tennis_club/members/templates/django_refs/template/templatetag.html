Django Template Tag Reference

Templatetag Template Tag

Templatetag Example

Output Django code without execute it:

{% templatetag openblock %} extends {% templatetag closeblock %}

Definition and Usage

The templatetag tag is used to display characters that are normally used to perform Django tasks.

{#

Each tag character, like {{ , {% and {# , has their own name, see below:

#}

Syntax

{ { % templatetag name % } }

Parameters

Value Description
name Required. A string specifying what to output.

Name Output
openvariables { {
closevariables } }
openblock { %
closeblock % }
openbrace {
closebrace }
opencomment { #
closecomment # }
