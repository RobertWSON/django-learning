Django Template Tag Reference

Firstof Template Tag

Use firstof when there is no empty values in an object

Firstof with no empty values Example

Return first of three variables (x, y, z), whose value is not empty or false:

{% firstof x y z %}

As you can see from this object example, all variables have a value.
As we are looking for first variable of object, it will be for x and that is Volvo.

Check out views.py to see content of x, y and z variables, for both examples.

Definition and Usage

The firstof tag returns first argument that is not an empty variable.

Empty variables can be an empty string "", or a zero number 0, or a boolean false.

Parameters

Value Description
var1 var2 var3 . Variables. The first variable that is not empty, 0, or false is returned.
If all variables are empty, then nothing is returned.
