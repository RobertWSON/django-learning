In our HTML template file we are using a Django filter command on our paragraph tag.
The filter we are using is upper, which is used here to show all paragraph text in upper case letters.
Finally we can use {% endfilter %} to close off our filter tag.

Definition and Usage

The filter tag allows you to run section of code through a filter and return it according to filter keyword(s).

To add multiple filters, separate keywords with pipe | character.

Syntax

{% filter keyword %}
...
{% endfilter %}

Parameters

The filter tag takes any of keywords discussed in filter section as parameter(s).
