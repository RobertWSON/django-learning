Django Template Tag Reference

Template Tag - filter

Filter Example

Output text in upper case:

Welcome Everyone!

{% filter upper %}

Have a great day!

{% endfilter %}

Definition and Usage

The filter tag allows you to run section of code through a filter and return it according to filter keyword(s).

To add multiple filters, separate keywords with pipe | character.

Syntax

{ % filter keyword % } ... { % endfilter % }
