Autoescape Off Template Example


Here we are demonstrating how Django Autoescape Off works for a variable that has an italics HTML tag with a heading.
In our views.py file, we have this heading variable Hello <i>my<i> World! and this is how it shows on our web page below.

{% autoescape off %}

{{ heading }}

{% endautoescape %}

With Django Autoescape off, you can see how italics HTML tags does not get escaped (converted back to text) and gets shown
as result of italics HTML tags.

This means that heading Hello my World! has my shown in italics.
