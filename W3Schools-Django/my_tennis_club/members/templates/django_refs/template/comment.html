Definition and Usage

The comment tag allows you to add comment sections that will be ignored by Django.

Comments can be used to make code more readable.

Comments can be used to prevent execution when testing code.

You can add an explanation to your comments to make them more understandable:

Comment Example (that has an explanation)

Add an explanation to your comments to make them more understandable.

Welcome Everyone!

{% comment %}

Greetings!

{% endcomment %}

Syntax

comment explanation {with % % inside brackets}

...

endblock {with % % inside brackets}

Parameters

Value Description
explanation Optional. A string, specifying an explanation for that particular comment.
