Autoescape On Template Example


Here we are demonstrating how Django Autoescape On works for a variable that has an italics HTML tag with a heading.
In our views.py, just like we did with Autoescape Off, we have same heading variable Hello <i>my<i> World!
and this is how it shows on our web page below.

{% autoescape on %}

{{ heading }}

{% endautoescape %}

When Django Autoescape was off with same heading variable, we saw that it showed my as italics with heading.

Now with Django Autoescape on we can see that italics HTML tags get escaped (converted from HTML) and shown as text.
This is how Django Autoescape On behaves when we have a variable with HTML tags.
