Definition and Usage

The if tag allows you to write conditional statements.

Use if statements to output a block of code if a condition is true.

You can use else or elif (short for "else if") to specify what to do when an if condition is false.

If using elif for a Heading Example

Display a third heading if none of conditions are true:

{% if myvar_elifheader == 1 %}

Hello!

{% elif myvar_elifheader == 2 %}

Welcome!

{% else %}

Greetings!

{% endif %} {# Show image of views.py and urls.py file #} {# #} {# Explain code in views.py file #}

In our HTML template file we are using a Django if command to ask if our myvar_elifheader variable has a value of 1.

Our myvar_elifheader variable does not have a value of 1, so now we can use an elif condition.
This elif condition starts off with an else part that creates a new condition for myvar_elifheader variable with an if part,
for checking if value is 2.
For this example myvar_elifheader variable does have a value of 2, so this means that we display Welcome! heading.

Say our example did not have a myvar_elifheader variable value of 1 or 2, then we have to use last else condition
and display Greetings! heading.

Finally we use { % endif % } to close off whole Django if statement.
