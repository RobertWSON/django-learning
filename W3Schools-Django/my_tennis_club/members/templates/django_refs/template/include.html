Definition and Usage

The include template tag allows you to include content from another template.

Place include tag exactly where you want content to be displayed.

This is useful when you have same content for many pages.

You can also send variables into template, by using with keyword:

Syntax

{#{ { % include template % } }#} { % include template % }

or

{#{ % include template with key=value % }#} { % include template with key=value % }

Parameters

Value Description
template Required. A filename of a template. Either a string or a variable.
key=value Optional. A variable name and value to send into include field.
Used together with a with keyword. You can have as many key/value pairs as you like.
