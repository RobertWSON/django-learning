Definition and Usage

The extends tag is used to specify that this template needs a parent template.

The extends tag takes one argument, which is name of parent template.

When a child template with a parent template is requested, django uses parent template as a "skeleton"
and fills it with content from child template, according to match block tags.

Above Example from a Parent Template Perspective

The parent template has blocks that fit the child template's blocks.
