Extends Template Tag

Use extends in Child Template to get data for parent template

Specify that this template relies on a parent template:

{#

#} Extends Example {#

Definition and Usage

The extends tag is used to specify that this template needs a parent template.

The extends tag takes one argument, which is name of parent template.

When a child template with a parent template is requested, django uses parent template as a "skeleton" and fills it with content from child template, according to match block tags.
