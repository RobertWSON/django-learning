Django Template Tag Reference

Template Tag - Spaceless

Spaceless Example

Remove space between HTML tags:

{% spaceless %}
    {% for x in fruits %}
  • {{x}}
    • {% endfor %}
{% endspaceless %}

Right-click result and view page source. Entire HTML code is written in one line.

Check out views.py to see what fruits variable looks like.

Definition and Usage

The spaceless tag is used to remove any space between tags, in code.

The spaceless tag removes any whitespaces,new lines and tabs.

Syntax

{ % spaceless % } ... { % endspaceless % }
