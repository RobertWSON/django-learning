With for a variable in a template Example

Create a variable in a template and use it:

{# {% include "django_refs/template/with_example1.html"%} #} {# I have found that we cannot use include for this example and it is better to use one view with #} {% with firstname="Stalikken" %}

Hello{{ firstname }}

{% endwith %} {# Image showing code in HTML Template file for this example #} {# Explain Django code in HTML file #}

You can see, instead of using firstname variable from views.py, we are actually using firstname variable within this HTML templates.

Definition and Usage

The with tag is used to create variables in Django templates.

This can be useful when you need to ask for same variable many times, like in a loop.

With in a For Loop Example

Use with tag to get length of fruits only one time:

{# {% include "django_refs/template/with_example2.html" %} #} {# I have found that we cannot use include for this example and it is better to use one view with #} {# Django code in this HTML Template File. #} {% with var=fruits|length %} {% for x in fruits %}

{{ x }} is one of {{ var }} fruits.

{% endfor %} {% endwith %} {# Show image of views.py file #} {# Explain code in views.py file #}

Check out views.py to see what fruits list looks like.
You can see that within our context we are using an array with a list of fruit items.

Note: Our object fruits here holds an array which holds all fruit items.

If you want to have this example within a web page that you are creating like I have, then within context, you can have
fruits object pickup a fruits array. Outside context you can show the array with its item names.
With this newer return method we are are using render to pickup request, template and context.

{# Show image of Django code in HTML file #} {# Explain Django code in HTML file #}

In HTML Template, first line of code is creating a variable var for an object and naming it fruits.
A length filter is then used for number of items within this object, which is 4.
Next we are using a for loop for fruits variable (object), where x represents an item in this object.
A looped Paragraph tag is used to show each item name using x and it shows var (total number of items in fruits array)
using a length filter for array length.
