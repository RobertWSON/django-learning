Definition and Usage

{% for x, y in mycar.items %}

The {{ x }} is {{ y }}.

{# Show image of Django code in HTML Template file #} {# #} {# Explain Django code in HTML file #}

In our HTML template file we are using a Django for command with our dictionary.
A Dictionary is different from an array in that they have key-value pairs, which can be called items.

The for loop will loop through key-value pairs for our properties of our dictionary.
We will show a property name like brand and property value like Ford where needed.
In our for loop, property name will be called x and property value will be called y.
To access each of our items within mycar dictionary, we can use mycar.items.
Finally we can use {% endfor %} to close off our for loop.

Syntax

{% for item in object %}
...
{% endfor %}

Parameters

Value Description
item Required. Variable name(s) that represent item(s) of
an iterable object.
object Required. An iterable object.
