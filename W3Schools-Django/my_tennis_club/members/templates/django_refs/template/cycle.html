Cycle for new color in for loop Example

Add a new color for each iteration in a for loop:

    {% for x in fruits %}
  • {{ x }}
    • {% endfor %}

As you can see from this array example, each fruit word changes color as there is 1 cycle for the whole array.

Check out views.py to see what fruits variable looks like, for both examples.

Definition and Usage

The cycle tag returns different values for different iterations in a loop.

The first iteration gets a first value, second iteration gets second value etc.

You can have as many values as you like.

If there are more iterations than values, cycle resets and starts at value 1:

Cycle - How it works with restarting

Cycle with reset on restart Example

    {% for x in fruits %}
  • {{ x }}
    • {% endfor %}

As you can see from this array example, there are 4 fruit words, but only 2 colors
The 2 colors can be used in 1 cycle, so the cycle resets on 3rd word Cherry and restarts on first color red.

Syntax

{# Need space below between a curly brace { and %, so that it can be displayed as text and does not think it is code #} {#{ % resetcycle name % } #} { % cycle arg1 by arg2 arg3 etc . % }

Parameters

Value Description
