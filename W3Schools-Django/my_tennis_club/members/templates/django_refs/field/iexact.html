Definition and Usage

The iexact lookup is used to get records with a specified value.

The iexact lookup is case insensitive, so this means that it does not matter if a name starts with a lower case or upper case letter.

For a case sensitive search, use the exact lookup.

SQL Equivalent

The SQL equivalent to the example above will be:

WHERE firstname = 'Emil';

Syntax

All Field lookup keywords must be specified with fieldname, followed by two underscore characters __ and a keyword:

fieldname__iexact='value'
