Django Field Lookup Tag Reference

Field Lookups - startswith

Startswith Example

Get all records where firstname starts with letter "S":

The queryset object:

{{ mymembers }}

Loop through the items:

{% for x in mymembers %} {% endfor %}
ID Firstname Lastname
{{ x.id }} {{ x.firstname }} {{ x.lastname }}

Definition and Usage

The startswith lookup is used to get records that start with a specified value.

The startswith lookup is case sensitive.

For a case insensitive search, use istartswith lookup.

SQL Equivalent

The SQL equivalent to the example above will be:

WHERE firstname LIKE 'S%';

Syntax

All Field lookup keywords must be specified with fieldname, followed by two(!) underscore characters __ and a keyword:

fieldname__startswith='value'
