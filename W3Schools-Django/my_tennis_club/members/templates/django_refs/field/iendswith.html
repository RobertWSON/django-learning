Definition and Usage

The iendswith lookup is used to get records that end with a specified value.

The iendswith lookup is case insensitive, so this means that it does not matter if a name ends with a lower case or upper case letter.

For a case sensitive search, use endswith lookup.

SQL Equivalent

The SQL equivalent to the example above will be:

WHERE firstname LIKE '%s';

Syntax

All Field lookup keywords must be specified with fieldname, followed by two underscore characters __ and a keyword:

fieldname__iendswith='value'
