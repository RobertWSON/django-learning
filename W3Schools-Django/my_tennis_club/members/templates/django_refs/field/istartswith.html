Definition and Usage

The istartswith lookup is used to get records that start with a specified value.

The istartswith lookup is case insensitive, so this means that it does not matter if a name starts with a lower case or upper case letter.

For a case sensitive search, use startswith lookup.

SQL Equivalent

The SQL equivalent to the example above will be:

WHERE firstname LIKE 's%';

Syntax

All Field lookup keywords must be specified with fieldname, followed by two underscore characters __ and a keyword:

fieldname__istartswith='value'
