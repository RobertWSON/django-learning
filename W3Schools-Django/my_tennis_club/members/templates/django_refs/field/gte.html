Django Field Lookup Tag Reference

Field Lookups - gte (greater than, or equal to)

Greater Than or Equal To Example

Get all records where id is 3 or larger:

The queryset object:

{{ mymembers }}

Loop through the items:

{% for x in mymembers %} {% endfor %}
ID Firstname Lastname
{{ x.id }} {{ x.firstname }} {{ x.lastname }}

Definition and Usage

The gte lookup is used to get records that are larger than, or equal to, a specified value.

For a greater than, but not equal to search, use the gt lookup.

SQL Equivalent

The SQL equivalent to the example above will be:

WHERE id >= 3;

Syntax

All Field lookup keywords must be specified with fieldname, followed by two(!) underscore characters __ and a keyword:

fieldname__gte=value
