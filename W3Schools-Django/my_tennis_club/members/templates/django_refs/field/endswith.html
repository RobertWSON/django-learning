Django Field Lookup Tag Reference

Field Lookups - endswith

Endswith Example

Get all records where firstname ends with letter "s":

The queryset object:

{{ mymembers }}

Loop through the items:

{% for x in mymembers %} {% endfor %}
ID Firstname Lastname
{{ x.id }} {{ x.firstname }} {{ x.lastname }}

Definition and Usage

The endswith lookup is used to get records that end with a specified value.

The endswith lookup is case sensitive.

For a case insensitive search, use iendswith lookup.

SQL Equivalent

The SQL equivalent to the example above will be:

WHERE firstname LIKE '%s%';

Syntax

All Field lookup keywords must be specified with fieldname, followed by two(!) underscore characters __ and a keyword:

fieldname__endswith='value'
