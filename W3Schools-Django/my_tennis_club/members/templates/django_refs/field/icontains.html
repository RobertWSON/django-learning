Definition and Usage

The icontains lookup is used to get records that contain a specified value.

The icontains lookup is case insensitive.

For a case insensitive search, use the contains lookup.

SQL Equivalent

The SQL equivalent to the example above will be:

WHERE lastname LIKE '%ref%';

Syntax

All Field lookup keywords must be specified with fieldname, followed by two underscore characters __ and a keyword:

fieldname__icontains='value'
