Definition and Usage

The lte lookup is used to get records that are less than, or equal to, a specified value.

For a less than, but not or equal to search, use lt lookup.

SQL Equivalent

SQL equivalent to example above will be:

WHERE id <= 3;

Syntax

All Field lookup keywords must be specified with fieldname, followed by two underscore characters __ and a keyword:

fieldname__lte=value
