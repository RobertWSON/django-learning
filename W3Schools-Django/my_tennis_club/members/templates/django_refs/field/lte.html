Django Field Lookup Tag Reference

Field Lookups - lte (less than, or equal to)

Less Than or Equal to Example

Get all records where id is 3 or less:

The queryset object:

{{ mymembers }}

Loop through the items:

{% for x in mymembers %} {% endfor %}
ID Firstname Lastname
{{ x.id }} {{ x.firstname }} {{ x.lastname }}

Definition and Usage

The lte lookup is used to get records that are less than, or equal to, a specified value.

For a less than, but not equal to search, use the lt lookup.

SQL Equivalent

The SQL equivalent to the example above will be:

WHERE id <= 3;

Syntax

All Field lookup keywords must be specified with fieldname, followed by two(!) underscore characters __ and a keyword:

fieldname__lte=value
