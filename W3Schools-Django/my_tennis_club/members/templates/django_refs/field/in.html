Django Field Lookup Tag Reference

Field Lookups - in

In Example

Get all records where firstname is one of values in a list:

The queryset object:

{{ mymembers }}

Loop through the items:

{% for x in mymembers %} {% endfor %}
ID Firstname Lastname
{{ x.id }} {{ x.firstname }} {{ x.lastname }}

Definition and Usage

The in lookup is used to get records where a value is one of values in an iterable (list, tuple, string, queryset).

The in lookup is case sensitive.

SQL Equivalent

The SQL equivalent to the example above will be:

WHERE firstname IN ('Tobias', 'Linus', 'John');

Syntax

All Field lookup keywords must be specified with fieldname, followed by two(!) underscore characters __ and a keyword:

fieldname__in=[value1, value2, value3, ...]
