Definition and Usage

The in lookup is used to get records, where a value is one of values in an iterable.
An iterable can be a list, tuple, string or queryset.

The in lookup is case sensitive.

SQL Equivalent

The SQL equivalent to the example above will be:

WHERE firstname IN ('Tobias', 'Linus', 'John');

Syntax

All Field lookup keywords must be specified with fieldname, followed by two underscore characters __ and a keyword:

fieldname__in=[value1, value2, value3, ...]
