Django Admin

Django admin is is a great tool in Django that uses CRUD as a user interface for all models.

CRUD stands for Create Read Update Delete
This means that we can create a record within a model using a user interface (Django Admin), which is a built-in program that helps manage records within models for our database.
A record can be read for data that it has, updated through changes and deleted if needed all through Django Admin.

Django Admin is free and comes ready-to-use with Django:

URL goes to Django Admin log in page because of path shown below in urls.py file of your Django project:

my_tennis_club/my_tennis_club/urls.py

urlpatterns[] list takes requests going to admin/ path and sends them to admin.site.urls, which is part of a built-in application that comes with django.
This Django built-in application contains a lot of functionality and user interfaces, with one of them being a log-in user interface.
