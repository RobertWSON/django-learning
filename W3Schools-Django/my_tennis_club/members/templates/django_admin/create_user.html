In Django Admin we can create, read, update and delete groups and users, but where is Members model ?

Missing Django Model

We have logged into Django Admin with a user name, but now we have a missing Members model.

There is nothing wrong with Members model missing at this stage, because we have to tell Django which models should show in Django admin interface.

We will learn next how to include Members model in our Django Admin.
