First CSS Example Didn't Work

Just testing ? If you want to play around and not deploy your work, you can set DEBUG=TRUE
in settings.py file and example aboce will work.

Plan to deploy ? If you plan to deploy your work, you should set DEBUG=FALSE in settings.py file.
Example above will fail because Django has no built-in solution for serving static files, but there are other ways
to serve static files. You will understand how this is done with Whitenoise on next page.

ALLOWED_HOSTS

When using DEBUG = False you have to specify which host name(s) are allowed to host your work.
You choose '*' , which means any address are allowed to host this site. This should be changed into
a real domain name when you deploy your project to a public server.
