Deploy with Elastic Beanstalk

Elastic Beanstalk

In Amazon Web Services, navigate to Elastic Beanstalk application, as we did when we chose a provider
and click "Create Application" button:

{# #}

Create EB Application

Once you have clicked "Create Application" button, you will be taken to this page below, where
you can give your Django project a name. I will name it "my_first_django_project":

{# #}

Choose Platform

Scroll down until you see "Platform" section and choose "Python", with recommended version:

{# #}

Upload .zip File

Next, scroll down to next "Application code" section and choose "Upload your code".

Under "Source code origin section", Click on "Choose file" button, navigate to .zip file created earlier and upload:

{# #}

As you can see below .zip file is uploaded and now wer are ready to deploy!

{# #}

Deploy

Click "Create Application" button to start deploying.

{# #}

Waiting

Deployment will take a few minutes, so please be patient, wait and let it go through its process.

{# #}

Success!

Finally project is live and you can view it by clicking link below Myfirstdjangoproject-env header,
or type host address in your browser:
