Create AWS Account

Why AWS?

There are many server providers out there that can host Django projects and PostgreSQL databases.

We will use Amazon Web Services (AWS) platform, mainly because they offer a free solution that can
host both Django projects and PostgreSQL databases and All that is needed is an account.


Note: You can choose whatever server provider you like, they all give same result, but they
will have some provider-specific settings that you should be aware of.

AWS

Go to aws.amazon.com, and create an account:

Add RDS Service

Once you have an AWS account, you can start creating a database.

We will use a database service at AWS, called RDS.

RDS stands for Relational Database Service and it can be found in search field by typing RDS
and then under Service click on RDS to start it's service.
