Create PostgreSQL Database

Inside RDS service, create a postgresql database, either by navigating to Database section or just
clicking "Create database" button:

PostgreSQL Database Settings

Once you have started creating a database, you will be given some choices for type and settings of your databases.

To add a PostgreSQL database to your Django project, choose following options:

{# #}

Database Engine Method:

{# #}

Database Free Tier Template:

{# #}

Database Settings (Name, Username and Password):

You can choose any name, username, and password.

{# #}

Database Configuration:

Keep default instance configuration.

{# #}

Database Storage:

It can be a good idea to enable storage autoscaling by clicking on check box when a project is large.
For this project it is not necessary though, as members database does not have a large amount of data.

{# #}

Database Connectivity:

We make a connection with database by granting public access and creating a new security group.
Make sure public access is given by clicking yes and give security group a name.
It can be called what you like, but a suitable name for this one is "w3-django-project".

{# #}

Database Authentication:

Keep database default password authentication.

{# #}

Database Monitoring:

Keep database default monitoring.

{# #}

Click Create database:

Note monthly costs and that our free tier is available for 12 months with 20 Gigabytes of general
purpose storage.

Click on Create database button
This will take a few minutes, so please be patient, but when it is finished, you will have a new
PostgreSQL database.
