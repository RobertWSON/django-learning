Django - Create Virtual Environment

Virtual Environment

It is suggested to have a dedicated virtual environment for each Django project because it helps to manage dependencies
and avoid conflicts with system-level Python installations.
{# found from google search "why is a virtual environment needed in django" #} Virtual environments are useful ways of ensuring that correct package/ library versions are consistently used
every time software runs.
{# found from builtin.com/data-science/python-virtual-environment #} One way of managing a virtual environment is venv and documentation (included in python) can be found from
docs.python.org/3/tutorial/venv.html .

On this Django learning website we will call our virtual environment myworld, but you can decide
on a name you want to give it, it is your choice.
