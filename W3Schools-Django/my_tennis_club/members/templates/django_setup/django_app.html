Django Create App

What is an App ?

An app is a web application that has a specific meaning in your project, like a home page, a contact form, or a database.

In this Django learnining website we will create an App that allows us to list and register members in a database.
This App will also take you through how to use Django, with lots of examples.

But first, let's just create a simple Django app that displays "Hello World!".

Create Django App

We will name this app members .

If your server is running, turn it off, so that you are able to write commands.
On Windows use [control][break] and on Mac or Unix use [control][c].

If you are not in my_tennis_club folder of your Django project, use cd my_tennis_club to navigate to this folder
where you want to store this app.

{# Now we can run py manage.py startapp members command in command prompt to create our app. #} Now in our command line, on Windows we can run py manage.py startapp members command or on a Mac we can use
