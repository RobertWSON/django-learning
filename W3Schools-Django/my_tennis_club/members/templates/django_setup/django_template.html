Change Django Settings

To be able to work with more complicated stuff than "Hello World"!, we have to tell Django that a new app is created.

This is done in settings.py file in my_tennis_club/my_tennis_club folder.

In INSTALLED_APPS[] add members app like this, shown below on line 40:

my_tennis_club/my_tennis_club/settings.py

{# Image showing members added as an app within INSTALLED_APPS of settings.py file #}

As we have made a change in settings for this application, we need to bring in tools needed to help setup this application.
For us to do this, we need to make a migration by running a migrate command :

On Windows use py manage.py migrate
On my Mac or Unix use python manage.py migrate and you should see output like this:
