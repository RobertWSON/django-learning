#}

URL stand for uniform resource locator and it is a complete web address used to find a particular page on internet.
It uses a reference to a resource (a web page) located on a computer network (server) and has a method for getting that.
Web bowsers show URL of a web page in an address bar above a page.
A URL can have a form http://www.django.com/setup.html, with http as a hypertext transfer protocol, a hostname www.example.com
and a file name setup.html .
A URL thats starts with https:// shows that you are on a secure site.
